Smart Hospital Queue & Bed Management System
A desktop application built in Java + JavaFX that simulates real-time patient flow management in a hospital — from registration to discharge.
What it does:

🔹 Auto-prioritizes critical patients using a Min-Heap based Emergency Queue

🔹 Manages non-urgent patients with a FIFO OPD Queue

🔹 Tracks 50 hospital beds on a live, color-coded 2D grid

🔹 Maintains per-patient history logs using a Linked List

🔹 Supports undo of the last action via a Stack

🔹 Looks up patients in O(log n) using a Binary Search Tree

🔹 Manages department directory with a self-balancing AVL Tree

🔹 Sorts patient records using Bubble, Selection, and Insertion Sort
Tech Stack: Java 17 · JavaFX · NetBeans
This project was built as a hands-on application of core Data Structures — BST, AVL Tree, Heaps, Queues, Stacks, Linked Lists, 2D Arrays, and sorting algorithms — solving a real-world problem instead of just theory.
